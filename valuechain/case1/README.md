# 案例1-机械加工厂
机械加工厂的业务分为：  
* 工艺核算导入及报价
* 接单生产  

当前主要讨论接单生产。

## 1 核心流程
整理核心步骤如下：  
* 接收订单需求  
* 完成订单计划&产生采购需求  
* 发出采购订单  
* 接收采购物料  
* 下单生产
* 物料领取&记账  
* 产量登记
* 产品入库
* 发货  

日常常规任务：  
* 工资汇总  
* 其它采购
